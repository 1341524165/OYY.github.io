### 基本数据类型

#### 1.数据类型转换

byte < short < int < long < float < double  
**·** 其中任意两个都可以相互转换

#### 2.复合数据类型

eg.

```java
class Student
{
int age;
char gender;
}
```

#### 3.引用类型变量

eg.

**错误的：**

```java
Student zhangsan;
zhangsan.age = 21;
```

正确的应为：

```java
Student zhangsan;
zhangsan = new Student(); //构造方法
zhangsan.age = 21;
```
