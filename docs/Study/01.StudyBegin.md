### 3.发布多个 blog

:::tip 发布**多个 blog**
在 config.js 中声明，如下所示：

```js title="/static/docusaurus.config.js"
plugins: [
    [
      '@docusaurus/plugin-content-blog',
      {
        /**
         * 多实例插件必填。
         */
        id: 'second-blog',
        /**
         * 你的网站上博客的 URL 路由。
         * *请务必不要*添加末尾斜杠。
         */
        routeBasePath: 'my-second-blog',
        /**
         * 相对于站点目录的文件系统路径。
         */
        path: './my-second-blog',
      },
    ],
  ],
```

:::
