### 1.docusaurus博客基础
:::tip md文件**文字换行**
`双空格+换行打字`
:::

:::info **```**（下称点点点）的使用
`点点点`后加要highlight的语言类型，换行copy你的代码，再换行用`点点点`收尾【相当于pre标签】
``` js
var handsome
```
:::

:::tip 发布**多个blog**
在config.js中声明，如下所示：
``` js
plugins: [
    [
      '@docusaurus/plugin-content-blog',
      {
        /**
         * 多实例插件必填。
         */
        id: 'second-blog',
        /**
         * 你的网站上博客的 URL 路由。
         * *请务必不要*添加末尾斜杠。
         */
        routeBasePath: 'my-second-blog',
        /**
         * 相对于站点目录的文件系统路径。
         */
        path: './my-second-blog',
      },
    ],
  ],
```
:::
